# Índice dos Agentes Voyu

> Módulos especializados para diferentes demandas. Ative o agente certo para cada tarefa.

---

## COMO USAR OS AGENTES

1. Identifique qual tipo de demanda você tem
2. Abra o arquivo do agente correspondente
3. Use as instruções do agente como prompt ao iniciar a conversa com o assistente
4. O agente vai responder dentro do seu escopo e formato ideal
