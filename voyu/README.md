## COMO USAR OS AGENTES

Para cada demanda, identifique o agente certo:

| Demanda | Agente |
|---------|--------|
| "Preciso de uma legenda para o Instagram" | Agente 01 |
| "Quero revisar se esse post está no tom certo" | Agente 02 |
| "Quero melhorar a home do site" | Agente 03 |
| "Quero revisar essa proposta antes de enviar" | Agente 04 |
| "Quero pensar a estratégia de marketing do próximo trimestre" | Agente 05 |
| "Preciso de um script para responder a essa objeção" | Agente 06 |

Abra o arquivo do agente, copie o **PROMPT DE ATIVAÇÃO**, cole no início da conversa, e descreva sua demanda no campo `[DESCREVA AQUI]`.

## ALIMENTAR O SISTEMA

O sistema fica melhor conforme você adiciona informações. Priorize atualizar:

1. `CEREBRO.md` — ticket médio real, margem real, principais destinos, volume mensal
2. `BRAND/cliente-ideal-ICP.md` — perfis reais de clientes com dados reais
3. `SOCIAL-MEDIA/calendario-editorial.md` — o que funcionou, o que não funcionou
4. `COMERCIAL/estrategias-de-margem.md` — margens reais por categoria
